# ⚒ ReadmeCraft
Craft stunning README.md files for your GitHub projects in seconds — no sign-up, no build step, 100% in your browser.

ReadmeCraft is a beautiful, fully client-side README generator built with vanilla **HTML, CSS, and JavaScript**. Fill in a few fields, toggle the sections and badges you want, watch the live preview update in real-time, and export your polished `README.md` with one click.

---
## 🎬 Demo

> Open `index.html` in your browser to launch the app instantly — no installation needed.

---

## ✨ Features

- ⚡ **Instant Live Preview** — Watch your README take shape as you type
- 🧩 **10 Modular Sections** — Features, Installation, Usage, API, Screenshots, Contributing, Tests, Roadmap, Acknowledgments, License
- 🏷️ **8+ Smart Badges** — Auto-generated Shields.io badges for license, version, stars, forks, issues, PRs, build status, contributors
- 📑 **Auto Table of Contents** — Generated based on the sections you select
- 👁️ **Dual View Modes** — Toggle between rendered preview and raw markdown
- 📋 **One-Click Copy** — Copy your markdown to clipboard instantly
- ⬇ **Download as `.md`** — Export a ready-to-commit README.md file
- 🌙 **Dark Mode** — Easy on the eyes with a single click
- 💾 **Auto-Save** — Your work is preserved in localStorage between sessions
- 📱 **Fully Responsive** — Works seamlessly on desktop, tablet, and mobile
- 🔒 **100% Private** — No servers, no tracking, your data never leaves your device

## 🚀 Usage

### Option 1: Open directly

Simply double-click `index.html` or drag it into your browser.

### Option 2: Serve locally (recommended)

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx serve

# Using PHP
php -S localhost:8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

### Generating a README

1. Scroll to the **Generator** section
2. Fill in your project details in the **📝 Basics** tab
3. Toggle desired sections in the **🧩 Sections** tab
4. Pick badges in the **🏷️ Badges** tab
5. Click **📋 Copy** or **⬇ Download** to grab your `README.md`
